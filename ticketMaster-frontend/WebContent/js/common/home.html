Modern Business Features

The Modern Business template by Start Bootstrap includes:

  • Bootstrap v3.2.0
  • AngularJS v1.3.15
  • jQuery v1.11.0
  • Font Awesome v4.1.0
  • Unstyled page elements for easy customization
  • 17 HTML pages
  • 3 directives: portfolio, menu and carousel

This template can be easily changed for any kind of project. Header and footer was divided into directives. Upper menu is a another directive inside header and its loaded by a Json. Supports until 1 level of internal menu, can be extended for more levels if needed.
